# Sobre o Projeto

Este é um projeto básico de iniciação ao desenvolvimento web construído com as tecnologias fundamentais do front-end: HTML, CSS e JavaScript. Ele tem como objetivo reforçar conhecimentos de:

- Estruturação de páginas 
- Estilização visual
- Manipulação de lógica e interatividade com JavaScript  
- Geração de números aleatórios  
- Interação com o usuário  

Estruturação de páginas com HTML Estilização visual com CSS Manipulação de lógica e interatividade com JavaScript Trabalhar com geração de números aleatórios e interação com o usuário O jogo foi desenvolvido como parte de estudos/programação de lógica e interatividade simples para a web.

### Tecnologias Usadas

*   HTML
*   CSS
*   JavaScript

### Passo a passo:

1. Informe a **quantidade de números** a serem sorteados  
2. Informe o **menor** e o **maior valor** do intervalo  
3. Clique em **"Sortear"**  
4. Os números sorteados serão exibidos na tela  
5. Clique em **"Reiniciar"** para limpar os campos e poder sortear novamente
